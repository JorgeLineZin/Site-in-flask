Sobre o Projeto

Bem-vindo ao meu site! Este é um projeto de demonstração construído com o framework Flask em Python, projetado para mostrar um sistema de blog simples com autenticação de usuário, criação de posts e perfis de usuário.

O principal objetivo é aplicar e aprofundar os conhecimentos em desenvolvimento web back-end, incluindo o manuseio de bancos de dados com SQLAlchemy, gerenciamento de sessões de usuário com Flask-Login e a estruturação de uma aplicação Flask usando Blueprints.

O front-end foi estilizado para ser moderno e responsivo, utilizando CSS customizado com variáveis para um tema consistente e agradável, mesmo em dispositivos móveis.

Tecnologias Utilizadas

  • Back-end: Python, Flask, SQLAlchemy, Flask-Login
  • Front-end: HTML, CSS, JavaScript
  • Banco de Dados: SQLite (padrão)
  • Estrutura: Blueprints do Flask para organização
